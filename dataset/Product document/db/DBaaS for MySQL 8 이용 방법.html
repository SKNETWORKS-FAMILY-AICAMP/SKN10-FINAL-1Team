DBaaS for MySQL 8 이용 방법

  • DBaaS for MySQL 8은 kt cloud 공공 포탈(웹) 메인 페이지에서 ‘공공 kt cloud 상품 -> DB’메뉴를 통해 소개되어 있습니다.



  • kt cloud 콘솔에 로그인 하신 후, 아래와 같이 좌측 상단의 All Services 메뉴를 클릭하여
  • DBaaS for MySQL 8 사용 신청 후에 이용하실 수 있습니다.



  • 서비스 신청 완료 후, 콘솔 좌측 메뉴에 DBaaS for MySQL 8 서비스가 표시됩니다.
  • 아래와 같이 현재 총 7개의 주요 Feature 에 대한 하위 메뉴를 제공하고 있습니다.
  • All Service > DB > DBaaS for MySQL 8을 선택하여 콘솔 좌측 메뉴에 서비스를 추가할 수 있습니다.
