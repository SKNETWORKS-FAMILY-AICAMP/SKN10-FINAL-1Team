상품 개요

목적/용도

DBaaS for PostgreSQL은 PostgreSQL 데이터베이스를 편리하게 배포하고 운영할 수 있는 서비스입니다.
Master/Replica구조를 자동으로 구성하여 고가용성과 읽기 부하 분산을 제공합니다.
또한 UI로 간편하게 구조 변경이 가능하며, 실수를 예방하고 안정적으로 DB 서비스를 이용할 수 있는 다양한 관리 기능을 제공합니다.


서비스 특장점

ㅇ UI 기반 설정 : UI 기반의 편리한 DB 구조 변경 (Standalone ←→ Master/Replica)
ㅇ 고가용성 : Master/Replica 구조 제공으로 고가용성 및 Read 부하 분산 제공
ㅇ 안정성 : PostgreSQL 장애 발생 시, 자동 Fail Over/Fail Back 수행
ㅇ 파라미터 그룹 관리 : 복잡한 설정을 간단히 관리하는 파라미터 그룹 관리 기능
ㅇ 백업 기능 : 데이터 유실을 최소화 하기 위한 다양한 Backup 기능 제공
ㅇ 모니터링 : 모니터링, 알람 기능 제공


서비스 구조/원리

DBaaS for PostgreSQL은 Server 상품과 동일하게 VM 형태의 DB Instance로 고객 네트워크 내에 생성됩니다.
DB Instance에 대한 모든 생성/변경/삭제/기타 관리 사항은 DBaaS for PostgreSQL 서비스 콘솔에서 이용 가능합니다.
