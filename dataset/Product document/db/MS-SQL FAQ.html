FAQ

Q. MS-SQL에서 제공하는 추가 기능을 사용할 때, 추가 요금이 발생하나요?

  • MS-SQL 내에 포함되어 있는 기능은 추가 요금이 발생하지 않습니다.

    • Q. MS-SQL Server는 이미지 및 스냅샷 생성 불가한 이유가 무엇인가요?

  • Windows Server 의 경우 Server 생성 후 최초 부팅 시 내부 모듈이 동작하여 IP, 호스트, 계정, 비밀번호 등을 초기화 해주는데 MS-SQL을 업데이트 해주지는 못합니다.
    만약 Windows + MS-SQL Server 자체를 이미지 or 스냅샷으로 만들게 되면 MS-SQL 서버는 이전 Server의 정보를 갖고 있기 때문에 DB 정보가 모두 깨져버립니다.

  • 따라서 MS-SQL Server를 이전하려는 경우에는 신규 Server 를 생성하고, 기존 Server 내에서 DB를 복제하여 다른 서버로 옮기는 방법으로 권고 드립니다.
