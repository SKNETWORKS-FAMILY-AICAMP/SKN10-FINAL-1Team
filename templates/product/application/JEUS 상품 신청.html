JEUS 상품 신청

서버 생성

JEUS 상품은 Computing 상품 신청 메뉴에 위치하며 상품 신청 후 JEUS 이용을 위한 상세한 세팅법은 하단의 세팅법을 참고해주세요.
※ 만약 kt cloud를 처음 사용하는 사용자는 청약 절차(결제 정보 입력)를 거쳐야 합니다.
결제 정보는 최초 상품 신청 한 번만 입력하면, 향후 신청하는 모든 상품의 요금이 입력한 결제 정보로 청구됩니다.

아래 방법은 G1/G2와 D1간 차이가 있을 수 있습니다.
(1) 상품 청약 : All Services > Computing > JEUS





(2) 콘솔의 Server > Server > “서버 생성” 클릭




(3) 이미지는 Application > WAS > JEUS를 선택합니다.
(이외의 설정들은 Server 설정과 동일합니다.)





(4) “생성하기” 버튼을 누르면 서버가 생성됩니다.
JEUS 상품이 정상적으로 생성되면 아래 그림과 같이 서버리스트에서 확인할 수 있습니다.
(JEUS 서비스가 정상적으로 배포될 수 있도록, 생성하고 수분 뒤 접속하시기 바랍니다.)
