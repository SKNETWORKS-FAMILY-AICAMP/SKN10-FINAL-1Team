MS-SQL 상품 개요

목적/용도

Microsoft사의 MS-SQL DB를 월 비용으로 사용하실 수 있습니다. MS-SQL은 kt cloud Server Image(머신이미지) 형태로 제공되어지며, 이에 사용되는 VM은 Server 상품으로 이용 가능합니다.

kt cloud 의 MS-SQL Server 상품에서 제공하는 기능

1. 데이터베이스 엔진 서비스
2. SQL Server 복제
3. 검색을 위한 전체 텍스 및 의미 체계 추출
4. SQL Server 관리 도구 (SSMS)

추가로, kt cloud 에서 제공하는 MS-SQL은 서버 생성 후 자동 설치 되므로 MS-SQL 관리자 계정 패스워드는 VM의 패스워드로 설정 됩니다.
