AI SERV 가이드 - 폴더 관리

폴더 생성하기

AI SERV 사용자의 파일을 보존할 수 있도록 별도의 저장소를 지원합니다. 컨테이너의 기본 폴더는 컨테이너가 삭제되면 같이 삭제되므로, 보존해야하는 파일이 있을 경우 폴더에 저장하는 것이 좋습니다.
폴더는 Data&Model 탭에서 관리할 수 있습니다.


폴더 생성

폴더 생성 버튼을 눌러서 폴더를 원하는 이름으로 생성할 수 있습니다.
정상적으로 폴더 생성이 완료되면, 폴더 관리 리스트에서 정보를 확인할 수 있습니다.
생성된 폴더는 컨테이너 생성 시 ‘마운트할 폴더’ 목록에서 확인 가능하며, 여러 폴더를 동시에 마운트할 수도 있습니다.

폴더 삭제하기

폴더 리스트에서 폴더를 선택하고 ‘삭제’를 클릭하면 폴더가 삭제됩니다.



  • ※ 삭제된 폴더는 복구가 불가능하오니, 삭제 시 신중하게 선택하여 주시기 바랍니다.



    • 파일 브라우저 사용하기

    AI SERV에서는 웹 브라우저를 통해 원격 서버에 있는 파일 관리를 도와주는 파일 브라우저 프로그램을 지원합니다.
    특히 사용자의 로컬 머신에 있는 디렉토리를 업로드 할 떄에 유용하게 사용할 수 있습니다.
    현재 AI SERV에서는 파일 브라우저를 컨테이너 내에서 실행되는 어플리케이션 형태로 제공합니다.


    폴더 선택 후 파일 브라우저 버튼을 클릭해주세요.

  • 만약 파일 브라우저가 연결되지 않는다면, 브라우저의 팝업 허용 설정을 확인해주세요.

    • 파일 브라우저가 새창에서 열린 것을 확인할 수 있습니다. 선택한 폴더명을 파일 브라우저의 디렉토리에서 확인할 수 있습니다.
    파일 브라우저 창에서 디렉토리와 파일을 자유롭게 업로드하고 수정하고 삭제할 수 있습니다.



    사용자가 Data&Folder 탭에서 파일 브라우저 버튼을 클릭하면, AI SERV는 자동으로 파일 브라우저 전용 컨테이너를 하나 생성하게 됩니다.
    따라서, Container 탭에서 파일 브라우저 컨테이너가 조회되는 것을 확인할 수 있습니다. 이 컨테이너를 삭제하는 것은 사용자의 몫입니다.

  • 파일브라우저 컨테이너는 1 Core, 0.5 GB 사양으로 고정되며, 시간 당 63원의 요금이 부과됩니다.
  • GPU 열에 보이는 GPU 모델은 파일 브라우저 컨테이너가 생성된 노드 종류를 의미하는 것으로, 실제로 GPU가 할당된 것은 아닙니다.

    • 파일 브라우저 기본 사용법

    대부분의 파일 브라우저 작업은 매우 직관적이지만, 보다 자세한 가이드가 필요하실 경우 파일 브라우저 문서를 참조하십시오.
