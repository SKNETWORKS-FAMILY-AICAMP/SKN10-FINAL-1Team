AI SERV 가이드 - 볼륨 관리

볼륨 생성하기

AI SERV는 사용자의 파일을 보존할 수 있도록 별도의 저장소를 지원합니다. 컨테이너의 기본 폴더는 컨테이너가 삭제되면 같이 삭제되므로, 보존해야하는 파일이 있을 경우 볼륨의 data 폴더에 저장하는 것이 좋습니다.
볼륨은 메인 탭 좌측 상단의 볼륨 관리 버튼을 클릭하여 팝업 창에서 관리할 수 있습니다.

AI SERV에서는 컨테이너를 생성하기 위해 반드시 볼륨을 사용하도록 설정되어 있습니다.
생성된 볼륨은 이후 생성하는 모든 컨테이너에 자동으로 마운트됩니다. (data 폴더)


볼륨 생성

볼륨의 원하는 이름을 입력하고, 원하는 용량을 입력합니다.
볼륨 생성 시 신청 가능한 최소 용량은 1000GB이며, 추가 용량을 100GB 단위로 입력할 수 있습니다.
만약 1000GB만 사용하고자 하면, 추가 용량에는 0을 입력한 후 생성하기 버튼을 클릭합니다.

정상적으로 볼륨 생성이 완료되면, 볼륨 관리 리스트에서 전체 용량과 현재 사용량을 확인할 수 있습니다.

생성된 볼륨은 이후 생성하는 모든 컨테이너에 자동으로 마운트되며, 디렉토리 구조 상 ‘data’ 폴더에서 확인할 수 있습니다.


볼륨 용량 증설하기

볼륨 사용 중에 용량이 부족할 경우, 용량 증설도 가능합니다.
메인 탭에서 볼륨 관리 버튼을 클릭하여 볼륨 관리 팝업 창에 진입합니다.

볼륨 리스트에서 볼륨을 선택하고, 용량 증설을 클릭합니다.

변경할 용량을 입력하고 변경하기 버튼을 클릭합니다.
입력 수치는 현재 용량보다 커야하며 100GB 단위 최대 10,000GB까지 가능합니다.

볼륨 관리 리스트에서 용량이 정상적으로 변경된 것을 확인합니다.
용량 증설 시 Unmount 하지 않아도 증설할 수 있으며, 역시 자동으로 컨테이너에서 변경된 용량으로 인식됩니다.


볼륨 삭제하기

볼륨 리스트에서 볼륨을 선택하고 ‘삭제’를 클릭하면 볼륨이 삭제됩니다.




  • ※ 삭제된 볼륨은 복구가 불가능하오니, 삭제 시 신중하게 선택하여 주시기 바랍니다.
